To run the code, run jupyter with python environment installed and run the notebook of interested. The directory where notebooks reside needs to have "plots" directory. The environment needs to have packages numpy, matplotlib, cvxpy, cvxopt installed. If environment prompts to install any other packages, they can be installed using pip as well.
